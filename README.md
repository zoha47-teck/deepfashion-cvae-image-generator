# DeepFashion CVAE Image Generator

## Overview

This project implements a Convolutional Variational Autoencoder (CVAE) using PyTorch to generate synthetic fashion images from the DeepFashion dataset.

## Features

* Image preprocessing and normalization
* Convolutional VAE architecture
* Fashion image reconstruction
* Synthetic image generation
* Model saving and inference

## Technologies Used

* Python
* PyTorch
* Google Colab
* Matplotlib

## Results

The model successfully learned latent representations of fashion items and generated new synthetic clothing images.
